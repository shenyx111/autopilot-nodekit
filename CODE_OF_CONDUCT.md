# Code of Conduct

Be respectful, specific, and evidence-oriented.

This project deals with automation, long-running jobs, and local compute environments. When discussing problems, focus on reproducible behavior: what was run, what happened, and what evidence supports it.

Please do not use issues, discussions, or pull requests for harassment, personal attacks, spam, or unrelated promotion.

Do not share private data, credentials, proprietary logs, or real run directories in public threads. Redact sensitive details before posting.
